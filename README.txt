This code reproduces all figures from the manuscript titled "Comprehensive transcriptional map of primate brain development". All code was written in the R programming language and is contained within an R Studio project (https://www.rstudio.com/). Please download entire project and set the working directory to 'reports'. To generate figures, run scripts from 'src' or knit markdown documents in 'reports'.
